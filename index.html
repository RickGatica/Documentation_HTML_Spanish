Introducción

Esta página de documentación enumera todos los elementos HTML, que se crean mediante etiquetas. Están agrupados por función para ayudarlo a encontrar fácilmente el elemento que esté buscando. Se proporciona una lista alfabética de todos los elementos en la barra lateral de la página de cada elemento.

Raíz principal
Elemento Descripción
html El elemento HTML (html) representa la raíz (elemento de nivel superior) de un documento HTML, por lo que también se denomina elemento raíz. Todos los demás elementos deben ser descendientes de este elemento.
Metadatos del documento

Los metadatos contienen información sobre la página. Esto incluye información sobre estilos, scripts y datos para ayudar al software ( motores de búsqueda, navegadores, etc.) a usar y representar la página. Los metadatos para estilos y scripts se pueden definir en la página o enlazar a otro archivo que tenga la información.

Elemento Descripción
base El elemento HTML Base (base) especifica la URL base que se utilizará para todas las URL relativas en un documento.
head El elemento HTML Head (head) contiene información legible por máquina (metadatos) sobre el documento, como su título, scripts y hojas de estilo.
link El elemento HTML External Resource Link (link) especifica las relaciones entre el documento actual y un recurso externo. Este elemento se usa más comúnmente para vincular a hojas de estilo, pero también se usa para establecer iconos de sitios (iconos de estilo "favicon" e iconos para la pantalla de inicio y aplicaciones en dispositivos móviles), entre otras cosas.
meta El elemento HTML Metadata (meta) representa metadatos que no puede ser representado por otros elementos relacionados con los meta-HTML, como base, script, style o title.
style El elemento HTML Style (style) contiene información de estilo para un documento o parte de un documento.
title El elemento HTML Title (title) define el título del documento que se muestra en la barra de título de un navegador o en la pestaña de una página.
Raíz de seccionamiento
Elemento Descripción
body El elemento HTML Body (body) representa el contenido de un documento HTML. Solo puede haber un body elemento en un documento.
Sección de contenido

Los elementos de sección de contenido le permiten organizar el contenido del documento en partes lógicas. Utilice los elementos de sección para crear un esquema amplio para el contenido de su página, incluida la navegación de encabezado y pie de página, y elementos de encabezado para identificar secciones de contenido

Elemento Descripción
address El elemento HTML Address (address) indica que el HTML adjunto proporciona información de contacto para una persona o personas, o para una organización.
article El elemento HTML Article (article) representa una composición autónoma en un documento, página, aplicación o sitio, que pretende ser distribuible o reutilizable de forma independiente (por ejemplo, en sindicación).
aside El elemento HTML Target (aside) representa una parte de un documento cuyo contenido solo está relacionado indirectamente con el contenido principal del documento.
footer El elemento HTML Footer (footer) representa un pie de página para su contenido de sección más cercano o elemento raíz de sección. Un pie de página generalmente contiene información sobre el autor de la sección, datos de derechos de autor o enlaces a documentos relacionados.
header El elemento HTML Header (header) representa contenido introductorio, normalmente un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezados, pero también un logotipo, un formulario de búsqueda, un nombre de autor y otros elementos.
h1, h2, h3, h4, h5, h6 Los elementos HTML Heading (h1) a (h6) representan seis niveles de encabezados de sección. h1 es el nivel de sección más alto y h6 es el más bajo.
hgroup El elemento HTML Heading Group (hgroup) representa un encabezado de varios niveles para una sección de un documento. Agrupa un conjunto de h1 - h6 elementos.
main El elemento HTML Main (main) representa el contenido dominante del body de un documento. El área de contenido principal consiste en contenido que está directamente relacionado con el tema central de un documento o se expande sobre él, o la funcionalidad central de una aplicación.
nav El elemento HTML Navigation (nav) representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o hacia otros documentos. Los ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices.
section El elemento HTML Section (section) representa una sección independiente, que no tiene un elemento semántico más específico para representarla, contenida en un documento HTML.
Contenido del texto

Utilice elementos de contenido de texto HTML para organizar bloques o secciones de contenido colocadas entre las etiquetas de apertura y cierre del elementobody. Estos elementos identifican el propósito o la estructura de ese contenido y son importantes para la accesibilidad y el SEO del documento

Elemento Descripción
blockquote El elemento HTML Blockquote (blockquote) indica que el texto adjunto es una cita ampliada. Por lo general, esto se representa visualmente mediante sangría. Se puede proporcionar una URL para la fuente de la cita utilizando el atributo cite, mientras que se puede proporcionar una representación de texto de la fuente utilizando el cite elemento.
dd El elemento HTML Description Data (dd) proporciona la descripción, definición o valor del término anterior dt en una lista de descripción dl
div El elemento HTML Division (div) es el contenedor genérico de contenido de flujo. No tiene ningún efecto en el contenido o el diseño hasta que se le aplica estilo usando CSS.
dl El elemento HTML Description List (dl) representa una lista de descripción. El elemento incluye una lista de grupos de términos (especificados mediante el dt> elemento) y descripciones (proporcionadas por dd elementos). Los usos comunes de este elemento son implementar un glosario o mostrar metadatos (una lista de pares clave-valor).
dt El elemento HTML Description Term (dt) especifica un término en una descripción o lista de definiciones y, como tal, debe usarse dentro de un dl elemento.
figcaption El elemento HTML Figure Caption (figcaption) o título de figura representa un título o leyenda que describe el resto del contenido de su figure elemento principal.
figure El elemento HTML Figure (figure) (figura con título opcional) representa contenido autónomo, potencialmente con un título opcional, que se especifica mediante el figcaption elemento.
hr El elemento HTML Horizontal Rule (hr) representa una ruptura temática entre elementos a nivel de párrafo: por ejemplo, un cambio de escena en una historia o un cambio de tema dentro de una sección.
li El elemento HTML List Item (li) se utiliza para representar un elemento en una lista.
main El elemento HTML Main (main) representa el contenido dominante en el body de un documento. El área de contenido principal consiste en contenido que está directamente relacionado con el tema central de un documento o se expande sobre él, o la funcionalidad central de una aplicación.
ol El elemento HTML Ordered List (ol) representa una lista ordenada de elementos, normalmente representada como una lista numerada.
p El elemento HTML Paragraph (p) representa un párrafo.
pre El elemento HTML Preformatted (pre) representa texto preformateado que se presenta exactamente como está escrito en el archivo HTML.
ul El elemento HTML Unordered List (ul) representa una lista desordenada de elementos, normalmente representada como una lista con viñetas.
Semántica de texto en línea

Utilice la semántica de texto HTML en línea para definir el significado, la estructura o el estilo de una palabra, línea o cualquier fragmento de texto arbitrario.

Elemento Descripción
a El elemento HTML Anchor (a) (o elemento ancla), con su href atributo, crea un hipervínculo a páginas web, archivos, direcciones de correo electrónico, ubicaciones en la misma página o cualquier otra cosa que una URL pueda abordar.
abbr El elemento HTML Abbreviation (abbr) representa una abreviatura o un acronimo; el title atributo opcional puede proporcionar una expansión o descripción de la abreviatura.
b El elemento HTML Bold (b) se utiliza para llamar la atención del lector sobre el contenido del elemento, que de otro modo no se le concede un importancia especial
bdi El elemento HTML Bi-Directional Isolate (bdi) le dice al algoritmo bidireccional del navegador que trate el texto que contiene de forma aislada del texto que lo rodea.
bdo El elemento HTML Bi-Directional Override (bdo) anula la direccionalidad actual del texto, de modo que el texto que contiene se representa en una dirección diferente
br El elemento HTML Break (br) produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de líneas es significativa
cite El elemento HTML Cite (cite) se utiliza para describir una referencia a un trabajo creativo citado y debe incluir el título de ese trabajo.
code El elemento HTML Code (code) muestra su contenido con un estilo que pretende indicar que el texto es un pequeño fragmento de código informático.
data El elemento HTML Data (data) vincula un determinado contenido con una traducción legible por máquina. Si el contenido está relacionado con la fecha o la hora, se debe utilizar el elemento time.
dfn El elemento HTML Definition (dfn) se utiliza para indicar el término que se define dentro del contexto de una frase u oración de definición.
em El elemento HTML Emphasis (em) marca el texto que tiene énfasis. El elemento em se puede anidar, y cada nivel de anidamiento indica un mayor grado de énfasis.
i El elemento HTML Italic (i) representa un rango de texto que se diferencia del texto normal por alguna razón, como texto idiomático, términos técnicos, designaciones taxonómicas, entre otros.
kbd El elemento HTML Keyboard (kbd) representa un intervalo de texto en línea que denota la entrada textual del usuario desde un teclado, entrada de voz o cualquier otro dispositivo de entrada de texto.
mark El elemento HTML Mark (mark) representa el texto que está marcado o resaltado con fines de referencia o anotación, debido a la relevancia o importancia del pasaje marcado en el contexto adjunto.
q El elemento HTML Quotation (q) indica que el texto adjunto es una breve cita en línea. La mayoría de los navegadores modernos implementan esto rodeando el texto entre comillas.
rb El elemento HTML Ruby Base rb se utiliza para delimitar el componente de texto base de una ruby> anotación, es decir, el texto que se está anotando.
rp El elemento HTML Ruby Fallback Parenthesis (rp) se utiliza para proporcionar paréntesis alternativos para los navegadores que no admiten la visualización de anotaciones ruby utilizando el ruby elemento.
rt El elemento HTML Ruby Text (rt) especifica el componente de texto ruby de una anotación ruby, que se utiliza para proporcionar información de pronunciación, traducción o transliteración para la tipografía de Asia oriental. El elemento rt siempre debe estar contenido dentro de un ruby elemento.
rtc El elemento HTML Ruby Text Container (rtc) abarca anotaciones semánticas de caracteres presentados en Ruby Base rb elementos utilizados dentro del ruby elemento. Los elementos pueden tener anotaciones de pronunciación (rt) y semánticas (rtc).
ruby El elemento HTML Ruby (ruby) representa pequeñas anotaciones que se representan encima, debajo o al lado del texto base, que generalmente se utilizan para mostrar la pronunciación de caracteres de Asia oriental. También se puede utilizar para anotar otros tipos de texto, pero este uso es menos común.
s El elemento HTML Strikethrough (s) representa el texto con un tachado o una línea que lo atraviesa. Utilice el s elemento para representar cosas que ya no son relevantes o que ya no son precisas. Sin embargo, s no es apropiado cuando se indican ediciones de documentos; para eso, use los elementos del y ins, según corresponda.
samp El elemento HTML Sample (samp) se utiliza para incluir texto en línea que representa la salida de muestra (o citada) de un programa informático.
small El elemento HTML Small (small) representa comentarios laterales y letra pequeña, como derechos de autor y texto legal, independientemente de su estilo de presentación. De forma predeterminada, muestra el texto dentro de él un tamaño de fuente más pequeño, como small x-small.
span El elemento HTML Span (span) es un contenedor en linea generico para redactar contenido, que no representa nada de forma inherente. Se puede usar para agrupar elementos con fines de estilo (usando los atributos class o id), o porque comparten valores de atributo, como lang.
strong El elemento HTML Strong Importance (strong) indica que su contenido tiene una gran importancia, seriedad o urgencia. Los navegadores suelen mostrar el contenido en negrita.
sub El elemento HTML Subscript (sub) especifica texto en línea que debe mostrarse como subíndice por razones exclusivamente tipográficas.
sup El elemento HTML Superscript (sup) especifica texto en línea que se mostrará como superíndice por motivos exclusivamente tipográficos.
time El elemento HTML Time (time) representa un periodo específico en el tiempo.
u El elemento HTML Underline (u) representa un espacio de texto en línea que debe representarse de manera que indique que tiene una anotación no textual.
var El elemento HTML Variable (var) representa el nombre de una variable en una expresión matemática o un contexto de programación.
wbr El elemento HTML Word Break (wbr) representa una oportunidad de ruptura de palabras, una posición dentro del texto donde el navegador puede opcionalmente romper una línea, aunque sus reglas de ruptura de línea no crearían una ruptura en esa ubicación.
Imagen y multimedia

HTML admite varios recursos multimedia como imágenes, audio y video

>
Elemento Descripción
area El elemento HTML Area (area) define una región de zona activa en una imagen y opcionalmente, la asocia con un enlace de hipertexto. Este elemento se usa solo dentro de un elemento map.
audio El elemento HTML Audio (audio) se utiliza para incrustar contenido de sonido en documentos. Puede contener una o más fuentes de audio, representadas mediante el src atributo o el source elemento: el navegador elegirá la forma más adecuada. También puede ser el destino de los medios transmitidos mediante un archivo MediaStream
img El elemento HTML Image (img) incrusta una imagen en el documento.
map El elemento HTML Map (map) se utiliza con elementos area para definir un mapa de imagen (un área de enlace en la que se puede hacer clic).
track El elemento HTML Track (track) se utiliza como hijo de los elementos multimedia audio y video. Le permite especificar pistas de texto cronometradas (o datos basados en el tiempo), por ejemplo, para manejar automáticamente los subtítulos.
video El elemento HTML Video (video) incorpora un reproductor multimedia que admite la reproducción de video en el documento. También puede utilizarlo para contenido de audio, pero el elemento audio puede proporcionar una experiencia de usuario más adecuada.
Contenido incrustado

Además del contenido multimedia habitual, HTML puede incluir una variedad de otro contenido, incluso si no siempre es fácil interactuar con el.

Elemento Descripción
embed El elemento HTML Embed )(embed) incrusta contenido externo en el punto especificado del documento. Este contenido lo proporciona una aplicación externa u otra fuente de contenido interactivo, como un complemento de navegador.
iframe El elemento HTML Inline Frame (iframe) representa un contexto de navegación anidado, incrustando otra página HTML en la actual.
object El elemento HTML Object (object) representa un recurso externo, que puede tratarse como una imagen, un contexto de navegación anidado o un recurso para ser manejado por un complemento.
param El elemento HTML Parameter (param) define parámetros para un elemento object.
picture El elemento HTML Picture (picture) contiene cero o más elementos source y un elemento img para ofrecer versiones alternativas de una imagen para diferentes escenarios de visualización / dispositivo.
source El elemento HTML Source (source) especifica varios recursos multimedia para los elementos picture, audio y video.
Scripting

Para crear contenido dinámico y aplicaciones web, HTML admite el uso de lenguajes de scripting, principalmente JavaScript. Ciertos elementos apoyan esta capacidad

Elemento Descripción
canvas El elemento HTML Canvas (canvas) se utiliza con el canvas scripting API o la WebGL API para dibujar gráficos y animaciones.
noscript El elemento HTML Noscript (noscript) define una sección de HTML que se insertará si un tipo de secuencia de comandos en la página no es compatible o si la secuencia de comandos está actualmente desactivada en el navegador.
script El element HTML Script (script) se utiliza para incrustar código o datos ejecutables; esto se usa normalmente para incrustar o hacer referencia a código JavaScript.
Demarcar ediciones

Estos elementos le permiten proporcionar indicaciones de que se han modificado partes específicas del texto

Elemento Descripción
del El elemento HTML Delete (del) representa un rango de texto que se ha eliminado de un documento
ins El elemento HTML Insert (ins) representa un rango de texto que se ha agregado a un documento.
Contenido de la tabla

Los elementos aquí se utilizan para crear y manejar datos tabulares

Elemento Descripción
caption El elemento HTML Caption (caption) especifica el título de una tabla.
col El elemento HTML Column (col) define una columna dentro de una tabla y se usa para definir semánticas comunes en todas las celdas. Generalmente se encuentran dentro de un elemento colgroup.
colgroup El elemento HTML Column Group (colgroup) define un grupo de columnas dentro de una tabla.
table El elemento HTML Table (table) representa datos tabulares, es decir, información presentada en una tabla bidimensional compuesta por filas y columnas de celdas que contienen datos.
tbody El elemento HTML Table Body (tbody) encapsula un conjunto de elementos fila tr de la tabla, lo que indica que comprenden el cuerpo de la table.
td El elemento HTML Table Data (td) define una celda de una tabla que contiene datos. Participa en el modelo de mesa.
tfoot El elemento HTML Table Foot (tfoot) define un conjunto de filas que resumen las columnas de la table
th El elemento HTML Table Heading (th) define una celda como encabezado de un grupo de celdas en la tabla. La naturaleza exacta de este grupo está definida por los atributos scope y headers.
thead El elemento HTML Table Head (thead) define un conjunto de filas que definen el encabezado de las columnas de la tabla.
tr El elemento HTML Table Row (tr) define una fila de celdas en una tabla. Las celdas de la fila se pueden establecer mediante una combinación de elementos td (celda de datos) y th (celda de encabezado).
Formularios

HTML proporciona una serie de elementos que se pueden usar juntos para crear formularios que el usuario puede completar y enviar al sitio web o la aplicación.

Elemento Descripción
button El elemento HTML Button (button) representa un botón en el que se puede hacer clic, que se utiliza para enviar formularios o en cualquier lugar de un documento para una funcionalidad de botón estándar accesible.
datalist El elemento HTML Data List (datalist) contiene un conjunto de elementos option que representan las opciones permitidas o recomendadas disponibles para elegir dentro de otros controles
fieldset El elemento HTML Fieldset (fieldset) se utiliza para agrupar varios controles y etiquetas label dentro de un formulario web.
form El elemento HTML Form (form) representa una sección de documentos que contiene controles interactivos para enviar informacion
input El elemento HTML Input (input) se utiliza para crear controles interactivos para formularios basados en web con el fin de aceptar datos del usuario; Se encuentra disponible una amplia variedad de tipos de datos de entrada y widgets de control según el dispositivo y el agente de usuario.
label El elemento HTML Label (label) representa un título para un elemento en una interfaz de usuario.
legend El elemento HTML Legend (legend) representa un título del contenido de su elemento padre fieldset.
meter El elemento HTML Meter (meter) representa un valor escalar dentro de un rango conocido o un valor fraccionario.
optgroup El elemento HTML Options Group (optgroup) crea una agrupación de opciones dentro de un elemento select.
option El elemento HTML Option (option) se utiliza para definir un elemento contenido en elementos select, optgroup o un datalist. Como tal, el elemento option puede representar elementos de menú en ventanas emergentes y otras listas de elementos en un documento HTML.
output El elemento HTML Output (output) es un elemento contenedor en el que un sitio o aplicación puede inyectar los resultados de un cálculo o el resultado de una acción del usuario.
progress El elemento HTML Progress (progress) muestra un indicador que muestra el progreso de finalización de una tarea, que normalmente se muestra como una barra de progreso.
select El elemento HTML Select (select) representa un control que proporciona un menú de opciones
textarea El elemento HTML Text Area (textarea) representa un control de edición de texto sin formato de varias líneas, útil cuando desea permitir que los usuarios ingresen una cantidad considerable de texto de forma libre, por ejemplo, un comentario en una revisión o formulario de comentarios.
Elementos interactivos

HTML ofrece una selección de elementos que ayudan a crear objetos de interfaz de usuario interactivos

Elemento Descripción
details El elemento HTML Details (details) crea un widget de divulgacion en el que la información es visible solo cuando el widget se cambia a un estado "abierto"
dialog El elemento HTML Dialog (dialog) representa un cuadro de diálogo u otro componente interactivo, como una alerta, un inspector o una subventana descartable.
menu El elemento HTML Menu (menu) representa un grupo de comandos que un usuario puede realizar o activar. Esto incluye tanto los menús de lista, que pueden aparecer en la parte superior de una pantalla, como los menús contextuales, como los que pueden aparecer debajo de un botón después de hacer clic en él.
summary El documento HTML Summary (summary) especifica un resumen, título o leyenda en un cuadro de divulgación del elemento details.
Componentes Web

Web Components es una tecnología relacionada con HTML que permite, esencialmente, crear y utilizar elementos personalizados como si fuera HTML normal. Además, puede crear versiones personalizadas de elementos HTML estándar

Elemento Descripción
slot El elemento HTML Slot (slot) hace parte de la suite de tecnología de componentes web, es un marcador de posición dentro de un componente web que puede llenar con su propio marcado, lo que permite crear árboles DOM separados y presentarlos juntos.
template El elemento HTML Template (template) es un mecanismo para mantener HTML que no se representará inmediatamente cuando se carga una página, pero se puede crear una instancia posteriormente durante el tiempo de ejecución mediante JavaScript.
